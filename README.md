# 🛩️ نظام إدارة خدمات الملاحة الجوية (ANSMS Pro)

## 📋 نظرة عامة

**نظام إدارة خدمات الملاحة الجوية (Air Navigation Services Management System - ANSMS Pro)** هو منصة رقمية متطورة مصممة خصيصاً لقطاع الطيران المدني، تهدف إلى تحويل إدارة الموارد البشرية في مجال مراقبة الحركة الجوية من العمليات اليدوية التقليدية إلى نظام رقمي متكامل وآمن.

## 🎯 الميزات الرئيسية

### 🔐 الأمان والحماية
- تشفير متقدم لكلمات المرور باستخدام BCrypt
- إدارة صلاحيات دقيقة حسب الأدوار (Admin, Controller, Employee)
- حماية من الهجمات الإلكترونية
- نظام مصادقة متقدم مع إدارة الجلسات

### 👥 إدارة الموارد البشرية
- قاعدة بيانات شاملة لمراقبي الحركة الجوية
- إدارة التراخيص والشهادات مع تنبيهات انتهاء الصلاحية
- تتبع المسار الوظيفي والتحويلات
- إدارة المشاريع والتدريب

### 📊 لوحة التحكم الاستراتيجية
- تحليل SWOT للمؤسسة
- مؤشرات الأداء الرئيسية (KPIs)
- إحصائيات تفاعلية مع الرسوم البيانية
- تقارير استراتيجية للقرارات الإدارية

### 🔔 نظام الإشعارات
- تنبيهات تلقائية لانتهاء صلاحية التراخيص
- إشعارات البريد الإلكتروني عبر SendGrid
- إدارة الإشعارات والتنبيهات
- تقارير الإشعارات المرسلة

## 🛠️ التقنيات المستخدمة

- **إطار العمل**: ASP.NET Core 8.0
- **لغة البرمجة**: C#
- **قاعدة البيانات**: Microsoft SQL Server
- **واجهة المستخدم**: HTML5, CSS3, JavaScript, jQuery
- **التقارير**: QuestPDF, EPPlus
- **الأمان**: BCrypt, Identity Framework
- **الإشعارات**: SendGrid

## 🚀 التثبيت والإعداد

### المتطلبات الأساسية
- .NET 8.0 SDK
- Microsoft SQL Server 2019 أو أحدث
- Visual Studio 2022 أو أحدث

### خطوات التثبيت

1. **استنساخ المشروع**
   ```bash
   git clone [repository-url]
   cd HR-Aviation
   ```

2. **إعداد قاعدة البيانات**
   - قم بتشغيل SQL Server
   - أنشئ قاعدة بيانات جديدة باسم `HR-Aviation`
   - قم بتشغيل سكربتات قاعدة البيانات من مجلد `Database_Scripts/`

3. **تحديث إعدادات الاتصال**
   - عدّل ملف `appsettings.json` بإعدادات قاعدة البيانات الخاصة بك

4. **بناء وتشغيل المشروع**
   ```bash
   dotnet restore
   dotnet build
   dotnet run
   ```

5. **الوصول للنظام**
   - افتح المتصفح على: `https://localhost:5001`
   - بيانات الدخول الافتراضية:
     - اسم المستخدم: `admin`
     - كلمة المرور: `123`

## 📊 قاعدة البيانات

### الجداول الرئيسية
- **ControllerUsers**: بيانات مراقبي الحركة الجوية
- **Employees**: بيانات الموظفين
- **Licenses**: التراخيص والشهادات
- **Certificates**: الشهادات التدريبية
- **Projects**: المشاريع والتدريب
- **Observations**: المراقبة والملاحظات
- **Airports**: المطارات والدول
- **Permissions**: نظام الصلاحيات
